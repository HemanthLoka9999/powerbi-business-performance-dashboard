# Power BI Business Performance Dashboard

![Dashboard Preview](<dashboard overview.png>)

## Overview

This project presents an interactive **business performance dashboard** built using Microsoft Power BI.

The dashboard analyzes **sales performance, customer purchasing behavior, and geographic revenue distribution** using the **Cleaned UCI Online Retail Dataset** sourced from Kaggle.

It demonstrates how transactional e-commerce data can be transformed into **clear business insights through KPI monitoring and interactive visualizations**.

---

## Dataset

Source: Kaggle – Cleaned UCI Online Retail Dataset
x`
The dataset contains e-commerce transaction records including:

- Invoice information  
- Product details  
- Customer purchases  
- Country-level sales  
- Order quantities and revenue  

This dataset is commonly used for **retail analytics and customer purchasing behavior analysis**.

---

## Key Business Metrics

The dashboard tracks the following KPIs:

- **Total Revenue:** \$10.66M  
- **Total Customers:** 4K  
- **Total Orders:** 21K  
- **Average Order Value:** \$514  
- **Repeat Purchase Rate:** 65.58%

These KPIs provide a high-level view of **business performance and customer activity**.

## Dashboard Analysis

### Sales Performance
- Monthly revenue trend visualization
- Helps identify seasonal patterns and revenue fluctuations

### Product Performance
- Top products ranked by revenue
- Highlights the most profitable items

### Geographic Revenue Distribution
- Revenue comparison across countries
- Identifies key markets contributing to sales

### Customer Purchasing Behavior
- Buying frequency analysis
- Repeat purchase rate tracking
- Average order value evaluation

---

## Insights

Key observations from the dashboard include:

- Revenue is highly concentrated in **United Kingdom and European markets**
- A **65% repeat purchase rate** indicates strong customer retention
- A small group of products contributes significantly to overall revenue
- Monthly trends reveal periods of increased purchasing activity
